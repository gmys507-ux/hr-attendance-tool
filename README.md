# HR 근태정리 자동화 도구

이삼오구 HR팀 전용 / 매월 파견직·계약직 근태 자동 정리

---

## 🚀 빠른 시작 (최초 1회 셋업)

### 1) Python 설치 (이미 있으면 건너뜀)
- https://python.org/downloads/ → Python 3.10 이상
- 설치 시 **"Add Python to PATH"** 체크 ✓ 필수

### 2) 도구 실행
- 이 폴더의 **`시작.bat`** 더블클릭
- 최초 실행 시 1~2분 (라이브러리 자동 설치)
- 브라우저가 자동으로 열리며 `localhost:8501` 표시됨

### 3) 종료
- 콘솔 창에서 `Ctrl+C` 또는 창 닫기

## 💼 매월 작업 흐름

### Step 1. 인원 변경 사항 반영 (필요 시만)
- 좌측 메뉴 **👥 인원 관리** 클릭
- 표에서 직접 편집 (행 추가/삭제/수정)
- **💾 변경사항 저장** 버튼
- → 자동 백업 생성

### Step 2. 근태정리 생성
- 좌측 **📊 근태정리 생성** 클릭
- 원본 파일 3종 업로드:
  1. 원티드 주차 집계 (이삼오구_..._.xlsx, 16컬럼)
  2. 원티드 일자 상세 (이삼오구_..._EI*.xlsx, 87컬럼)
  3. FLEX 일자 리포트 (선택)
- 4주차 시작일 선택
- **🚀 근태정리 생성** 버튼 클릭

### Step 3. 다운로드
- 결과 엑셀 자동 다운로드 + `outputs/` 폴더에도 저장
- 검증 결과 + 한도 위반 경고 화면에 표시

## 🌐 다른 PC에서 사용하기 (사내망 공유)

### 본인 PC를 서버로 사용
1. 본인 PC에서 `시작.bat` 실행
2. 콘솔에 표시되는 **Network URL** 확인 (예: `http://192.168.10.150:8501`)
3. 다른 HR 직원에게 그 URL 공유
4. 같은 사내망 안의 다른 PC에서 브라우저로 접속

### 주의사항
- 본인 PC가 켜져 있어야 함
- 같은 Wi-Fi/회사 네트워크 필요
- 방화벽에서 8501 포트 허용 필요할 수 있음

### Streamlit Cloud 배포 (영구 서비스)
- 추후 GitHub에 코드 올리고 Streamlit Cloud에 연결하면 회사 외부에서도 접속 가능
- 단, HR 데이터 보안 검토 필요

## 🔧 자주 묻는 질문

**Q1. 매월 인원이 바뀌면 어떻게 하나?**
A. 👥 인원 관리에서 직접 수정. 자동 백업되니 실수해도 복구 가능.

**Q2. 인원 명단을 Excel로 한꺼번에 받고 싶다면?**
A. 인원 관리 → "📥 일괄 가져오기/내보내기" 탭 → Excel 다운로드.

**Q3. 신규 ERP/시스템이 추가되면?**
A. `lib/attendance_builder.py` 수정 필요 (개발자 작업).

**Q4. 결과 파일이 이상하다면?**
A. data/history/ 폴더에서 실행 이력 확인.
   인원 관리 → 백업 이력에서 이전 명단으로 복구 가능.

**Q5. 노션과 연동되나?**
A. 현재 버전은 Excel 출력만. 노션 자동 입력은 후속 작업.

## 📞 문제 발생 시

- 실행 안 됨 → Python 설치 + PATH 확인
- 한글 깨짐 → 시작.bat 의 `chcp 65001` 부분 확인
- 라이브러리 오류 → `pip install -r requirements.txt --upgrade`
- 데이터 손실 → `data/backup/` 폴더에서 복구
